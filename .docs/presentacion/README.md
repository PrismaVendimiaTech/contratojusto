## Como usar deck.html

1. Abrir el archivo `deck.html` directamente en Chrome
2. Navegar con las flechas del teclado (izquierda/derecha) o con los botones en pantalla
3. Modo fullscreen: F11
4. Listo para proyectar, no necesita internet ni instalar nada
